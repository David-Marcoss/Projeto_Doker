- Tutorial execulção do projeto

1 - Criar container fila:
	- docker run --rm -d --hostname my-rabbit --name some-rabbit  -e RABBITMQ_DEFAULT_USER=guest -e RABBITMQ_DEFAULT_PASS=guest rabbitmq:3-management

2- Criar container Banco de dados:
	- docker run --name mydb -e MYSQL_ROOT_PASSWORD=123456 -v mysqlVolume:/var/lib/mysql -d mysql:latest

3- Criar tabela banco de dados
	- docker exec -i id_do_container_mysql mysql -u root -p123456 <./db/schema.sql

4- Criar Container Consulmidor:
	- entre na pasta docker-consulmidor e execulte os comandos
	. install.sh
	- docker build --rm . --tag docker-consumidor
	- docker run -it --rm  docker-consumidor

5- Criar Container Produtor
	- entre na pasta docker-produtor e execulte os comandos
	- . install.sh
	- docker build --rm . --tag docker-produtor
	- docker run -it -p 3000:5000 docker-produtor	

6- Acessar aplicação:
	http://localhost:3000
